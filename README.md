# 🦞 OpenClaw — Multi-Agent Workflow Generator

Build multi-agent AI workflows visually. Generate `SOUL.md`, `agent.md`, and `workflow.md` files for your OpenClaw project.

**Live:** [workflow.clawknot.com](https://workflow.clawknot.com)

## Local Development

```bash
npm install
npm run dev
```

## Deployment

Pushes to `main` auto-deploy via GitHub Actions to GitHub Pages.

## Custom Domain

DNS (Cloudflare): `CNAME workflow → <username>.github.io`
